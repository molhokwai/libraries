jQs

A (jQuery) plugin to simplify intensive to the pixel css styling & sizing...

To make to the pixel css styling & sizing easier by being able to rely on the plugin to generate the desired style by using the jQs class and intuitively writing, for example:

class="jQs maut w60% padho2% padv1% fs0.8em fwebl tdund fll"

Plugin candidate

This plugin contains 450+ lines of code, ~14 lines of Jquery code & the rest pure javascript. It is easily extensible to allow more (all) css properties in. Right now it allows the most common styling & sizing properties a css web designer uses.

Code Wise

It would be easy to refactor and greatly reduce the number of lines of code, while making it even more flexible. If you wish to do that, let me know, and I'll happily give you the details.
